# **App Name**: BetterPlugins Hub

## Core Features:

- Plugin Discovery: Browse, search, and filter Paper Minecraft plugins.
- Plugin Detail Pages: Detailed view for each plugin, including overview, versions, documentation, and changelog.
- Version Management: Allow users to manage and view multiple versions of Minecraft plugins, including filtering plugins by Minecraft version.
- User Authentication (Future): A system for users to create accounts and manage their favorite plugins and settings.
- AI Plugin Suggestion Tool: A tool which uses a generative AI to suggest plugins relevant to the user, based on plugin name and description
- Download & Installation: One-click plugin downloads with clear installation instructions.

## Style Guidelines:

- Background: Deep navy (#1A202C) with subtle pixel or grid texture.
- Primary text: White (#FFFFFF) for high contrast.
- Accent color: Emerald green (#2ECC71) for CTAs and highlights.
- Secondary accent: Soft blue (#3B82F6) for interactive elements.
- Headings: 'Poppins' (sans-serif) for a bold, modern look. Note: currently only Google Fonts are supported.
- Body: 'Inter' (sans-serif) for readability. Note: currently only Google Fonts are supported.
- Code blocks: 'Source Code Pro' (monospace) for clear code snippets. Note: currently only Google Fonts are supported.
- Custom Minecraft-themed icons.
- Dark theme with block-style design.
- Framer Motion for smooth transitions and hover effects.
